# Excel Export Examples & Use Cases

## üìä Overview

The Excel export feature transforms your Hearth inventory data into professional, business-friendly spreadsheets that can be used for a wide variety of purposes. This document provides examples and use cases for the exported data.

## üìÅ Sample Files

### **Sample Excel Export**
- **File:** `sample-hearth-inventory-export.xlsx`
- **Format:** Multi-sheet Excel workbook
- **Contents:** Complete sample inventory with 3 containers, 8 items, 5 tags, and 4 categories

### **Sample JSON Export**
- **File:** `sample-inventory-export.json`
- **Format:** Technical JSON format
- **Contents:** Same data in machine-readable format for developers

## üìã Excel Workbook Structure

### **Sheet 1: Summary**
- **Export metadata** - When exported, by whom, version info
- **Inventory overview** - Total counts of containers, items, tags, categories
- **Financial summary** - Total estimated value, items with active warranties
- **Container statistics** - Average items per container, distribution analysis

### **Sheet 2: Containers**
- **Container details** - Name, location, description
- **Item counts** - How many items in each container
- **Total values** - Sum of all item values per container
- **Timestamps** - When containers were created and last updated

### **Sheet 3: Items**
- **Item information** - Name, description, container location
- **Categories & tags** - Organization and classification
- **Financial data** - Purchase prices, dates, current values (optional)
- **Warranty info** - Warranty details and expiration dates (optional)
- **Image status** - Whether items have photos (optional)

### **Sheet 4: Tags**
- **Tag details** - Name, color coding
- **Usage statistics** - How many items use each tag
- **Sorted by popularity** - Most-used tags first

### **Sheet 5: Categories**
- **Category hierarchy** - Category names and paths
- **Usage counts** - How many items in each category
- **Sorted by usage** - Most-used categories first

## üìù Best Practices

### **Regular Exports**
- **Monthly exports** - Keep regular backups of your inventory data
- **Before major changes** - Export before reorganizing or moving
- **Annual reviews** - Use exports for yearly inventory assessments
- **Sharing preparation** - Export clean data before sharing with others

### **Data Organization**
- **Consistent naming** - Use clear, descriptive names for items and containers
- **Complete descriptions** - Include model numbers, brands, and key details
- **Regular updates** - Keep purchase prices and warranty info current
- **Photo documentation** - Add photos to items for visual reference

### **Security Considerations**
- **Sensitive data** - Be cautious when sharing files with financial information
- **File encryption** - Consider encrypting files with valuable asset information
- **Access control** - Limit who has access to detailed inventory exports
- **Regular cleanup** - Delete old export files from shared locations
