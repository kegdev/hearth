## üì¶ Container Management

### Creating Containers
Containers are like digital boxes, shelves, or storage areas where you organize your items.

#### How to Create a Container
1. **Navigate** to the Containers page
2. **Click** "+ Add Container" button
3. **Fill in Details**:
   - **Name** (required): e.g., "Kitchen Drawer", "Bedroom Closet"
   - **Description** (optional): Additional details about the container
   - **Location** (optional): Physical location like "Living Room", "Garage"
   - **Image** (optional): Upload a photo of the container
4. **Click** "Create Container"

#### Container Features
- **QR Codes** - Generate QR codes for physical container labels
- **Image Support** - Add photos to visually identify containers
- **Location Tracking** - Keep track of where containers are physically located
- **Item Counting** - Automatic count of items in each container

### Managing Containers
- **View Details** - Click "View Items" to see container contents
- **Edit Container** - Update name, description, location, or image
- **Delete Container** - Remove container and all its items (owners only)
- **Generate QR Code** - Create printable QR codes for physical containers

## üìã Item Management

### Adding Items
Items are the individual belongings you store in your containers.

#### How to Add an Item
1. **Go to** a container detail page or the Items page
2. **Click** "+ Add Item" button
3. **Fill in Basic Details**:
   - **Name** (required): e.g., "Winter Jacket", "Photo Album"
   - **Description** (optional): Additional details about the item
   - **Container** (required): Select which container holds this item
4. **Add Advanced Details** (optional):
   - **Purchase Price** - What you paid for the item
   - **Current Value** - What it's worth now
   - **Purchase Date** - When you bought it
   - **Condition** - New, Excellent, Good, Fair, Poor
   - **Brand & Model** - Manufacturer and model information
   - **Serial Number** - For electronics and valuable items
   - **Warranty Info** - Warranty details and expiration
5. **Upload Image** (optional): Add a photo of the item
6. **Add Tags** (optional): Categorize with custom tags
7. **Select Category** (optional): Choose from organized categories
8. **Click** "Add Item"

### Item Features
- **Detailed Metadata** - Track purchase price, current value, condition, warranty
- **Photo Support** - Visual identification with image uploads
- **Tag System** - Organize items with custom colored tags
- **Category System** - Hierarchical organization system

### Managing Items

## ü§ù Container Sharing

### Sharing Your Containers
Share containers with family, friends, or roommates for collaborative inventory management.

#### How to Share a Container
1. **Open** the container you want to share
2. **Click** the blue "Share" button (ü§ù icon)
3. **Search for User** - Enter the email address of the person you want to share with
4. **Select Permission Level**:
   - **View** - They can see the container and items but cannot make changes
   - **Edit** - They can add, edit, and delete items in the container
   - **Admin** - Full control including sharing with others
5. **Click** "Share Container"

#### Permission Levels Explained

**View Permission**
- See container details and location
- View all items in the container
- Access item photos and descriptions
- Generate QR codes
- Cannot make any changes

**Edit Permission**
- Everything from View permission
- Add new items to the container
- Edit existing item details
- Delete items from the container
- Upload and change item photos
- Cannot edit container details or share with others

**Admin Permission**
- Everything from Edit permission
- Edit container details (name, description, location)
- Share container with other users
- Manage sharing permissions
- Revoke access from other users
- Cannot delete the container (only owner can)

### Managing Shared Access
- **View Current Shares** - See who has access to your containers
- **Update Permissions** - Change someone's permission level
- **Revoke Access** - Remove someone's access to your container

### Accessing Shared Containers
When someone shares a container with you:

1. **Shared containers appear** on your Containers page with blue headers
2. **"Shared by [Owner Name]"** indicator shows who shared it
3. **Permission badge** shows your access level (View Only, Edit, Admin)
4. **All features work** according to your permission level

## üîç Global Search

### Quick Search (Navbar)
The search bar in the navigation allows instant searching across your entire inventory.

#### How to Use Quick Search
1. **Click** in the search bar or press **Ctrl/Cmd+K**
2. **Type** your search term (container names, item names, descriptions, etc.)
3. **View Suggestions** - Up to 8 results appear in real-time
4. **Click** on any result to navigate directly to that container or item
5. **Click** "View all results" for comprehensive search results

#### Search Features
- **Real-time Suggestions** - Results appear as you type
- **Cross-Inventory Search** - Searches both owned and shared containers
- **Multi-Field Matching** - Searches names, descriptions, brands, models, serial numbers
- **Visual Indicators** - Clear icons and shared content badges
- **Keyboard Shortcuts** - Ctrl/Cmd+K to focus search

### Advanced Search (Results Page)
For comprehensive search results with detailed presentation.

#### Search Capabilities
- **Container Search** - Find containers by name, description, or location
- **Item Search** - Find items by name, description, brand, model, or serial number
- **Shared Content** - Includes containers and items shared with you
- **Grouped Results** - Items organized under their parent containers
- **Direct Navigation** - Click to view container or item details

#### Search Tips
- Use specific terms for better results
- Try brand names, model numbers, or serial numbers
- Search works across both your items and shared content
- Use broader terms if you don't find what you're looking for

## üö® Troubleshooting

### Common Issues

#### Can't See Shared Container Items
- **Check Permissions** - Ensure you have at least View permission
- **Refresh Page** - Sometimes a refresh resolves loading issues
- **Contact Owner** - Ask the container owner to verify sharing settings

#### Search Not Working
- **Check Internet Connection** - Search requires internet connectivity
- **Try Different Terms** - Use broader or more specific search terms
- **Clear Browser Cache** - Sometimes cached data causes issues

#### Can't Share Container
- **Verify Email** - Ensure the email address is correct
- **Check User Status** - User must be approved by admin
- **Permission Level** - Ensure you have admin permission on the container

#### Mobile Issues
- **Update Browser** - Use the latest version of your mobile browser
- **Check Connection** - Ensure stable internet connection
- **Clear Cache** - Clear browser cache and cookies
